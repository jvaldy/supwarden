# Guide utilisateur

## 1. Se connecter

1. Ouvrir la page de connexion.
2. Saisir e-mail et mot de passe, ou utiliser Google OAuth.
3. En cas d'echec, consulter le message d'erreur affiche.

## 2. Gerer les trousseaux

- Creer un trousseau depuis la liste.
- Ouvrir un trousseau pour consulter ses elements.
- Ajouter et gerer les membres selon vos droits.
- Modifier les parametres depuis le bouton dedie.

## 3. Gerer les elements

- Ajouter un element (identifiant, mot de passe, notes, URL, champs personnalises).
- Utiliser les actions de copie rapide.
- Ouvrir le detail d'un element pour consulter ses informations.
- Modifier ou supprimer selon vos permissions.

## 4. Securite PIN

- Definir votre code PIN sur la page profil.
- Les secrets sensibles demandent le PIN selon la configuration actuelle du projet.
- Le masquage automatique des secrets suit le delai configure.

## 5. Import / export

- Importer des donnees en JSON ou CSV depuis le dashboard ou la liste des trousseaux.
- Exporter tout ou un trousseau specifique selon le contexte.
- Verifier les messages de validation apres import.

## 6. Messagerie

- Messagerie privee : choisir un contact et envoyer des messages.
- Messagerie de trousseau : discuter avec les membres autorises.
- Les notifications se mettent a jour automatiquement.
